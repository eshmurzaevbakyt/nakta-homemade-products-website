# NAKTA — Homemade Frozen Products

Website for NAKTA LLC, a family-owned homemade frozen food business based in Seattle, WA.

## About

NAKTA specializes in handmade Central Asian frozen foods including Manty, Dumplings, and Meatballs. All products are 100% Halal, made with clean ingredients, no additives, and non-GMO.

## Tech Stack

- HTML / CSS / JavaScript
- Hosted on Netlify
- EN / RU bilingual support

## Features

- Fully responsive (mobile + desktop)
- Bilingual EN / RU language toggle
- WhatsApp order integration
- Halal certificate modal
- Google Maps store locations
